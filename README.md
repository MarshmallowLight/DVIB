# Anonymous Repository for VBPR Code (DVIB Framework)

This repository provides a simplified implementation of the VBPR model with the key functions of the proposed **DVIB framework**. This implementation is shared as part of our commitment to transparency and reproducibility during the peer review process.

## **Description**

The DVIB framework integrates the Information Bottleneck principle with self-distillation to enhance the robustness and performance of multimodal recommender systems. This repository focuses on the main function of the VBPR model with the core components of our framework.

## **Features**

- **Noise Injection**: Adds Gaussian noise to intermediate representations for enhanced robustness.
- **Self-Distillation**: Ensures consistency across model layers, improving feature learning.
- **Reproducibility**: Simplified code for easy understanding and implementation.
